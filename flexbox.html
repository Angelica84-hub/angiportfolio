Flexbox


Flexbox er mest nyttigt til de små skærme og mobile enheder, da den er fleksibel og derfor tilpasser sig. Man kan ‘’smide” masser af information/text på en side og kan stadig holde en ren orden i layout. Det er en CSS style der hjælper os med at få styr på hvordan sections/diver vises.


Vi har delt opgaven og vi arbejdet hver på sin egne emne.Til sidste vi samledet vores data og begyndt at kode i html og css. Det var vores første stor opgave der skulle laves i html og css.Det er lige der, jeg så hvordan en website bygges op.
