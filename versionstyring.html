Versionstyring


Versionstyring er en koncept der hjælper os at holde styre på alle vores gamle filer vi har arbejdet på, ved at gemme de på en server. Når det er mere end en person der arbejder på file så er bedre at anvendes til versionstyring for at undgå fejl, for at være nemt at gå tilbage hvis alligevel vi opstår en fejl, og for en nemt samarbejde. Vi bruger versionstyring i alle typer it projekter. Alle dem der arbejder med it udvikling, iværksættere, store organizationer, osv, anvender sig til versionstyring. En af de mest udbredte måder blandt webudviklere er baseret på Git protokollen. Git er et versionstyringssystem. Den er oprindeligt skrevet at Linus Thorvalds for, at holde styr på udviklingen af Linux. Github er en server hvor man kan oprette Git reponsitories. Mange open source projekter hostes her.
